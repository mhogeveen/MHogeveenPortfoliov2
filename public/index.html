Over Mij

Mijn naam is Maarten Hogeveen. Ik ben een web developer/designer. Sinds mijn jeugd heb ik al een affiniteit gehad met technologie. Mijn moeder, die zelf ook een web developer en designer was, heeft mij van jongs af aan blootgesteld aan de online wereld en het werk wat er achter het web schuilt.

Mijn interesses hebben ook altijd mijn affiniteit met technologie gereflecteerd. Of het nou het repareren van computers, gamen, video bewerking of streamen was om maar een paar dingen te noemen. Naast deze technische interesses heb ik ook altijd veel voldoening gehaald uit het helpen en gelukkig maken van anderen. Het maken van websites is een prachtig manier om deze twee aspecten van mij te combineren.

Hieronder vindt u een representatie van aantal van mijn vaardigheden op het gebied van web development/design met daar onder mijn portfolio.

Portfolio

COLMAR ACADEMY
Bekijk de website

Het Colmar Academy project was een oefening in het maken van mijn eigen design keuzes voor een al bestaand wireframe. Daarnaast was het een oefening in het creëren van een responsieve, statische website vanaf de grond af aan.

De design en kleurenpalet van deze website moet het gevoel van een moderne institutie wekken door middel van een strak en simpele layout die tot zijn recht komt door een rustig en industrieel uitziend kleurenpalet.

CONQUER REMAKE
Bekijk de website

De Conquer template remake was een oefening in het perfect namaken van een bestaande website template. Het hele project moest gemaakt worden in HTML en CSS zonder gebruik te maken van JavaScript.

Het recreëren van de Conquer template zonder JavaScript was een goede oefening in problemen oplossen. De meest voor de hand liggende oplossing voor een specifiek probleem was niet altijd binnen de zelf opgelegde restricties voor dit project. Het begrijpen en erkennen van het feit dat iets niet mogelijk was, was een uitstekende manier om te leren verder te gaan wanneer iets ten koste gaat van productiviteit.

PURE
Bekijk de website

Het maken van een website voor de fictieve koffiezaak Pure is een leerervaring geweest in het creëren van een website met meerdere pagina's die interactief en responsief is.

Dit project begon als een kopie maken van de Foundry template gemaakt door Medium Rare. Tijdens het ontwikkelingsproces begon de website echter een eigen identiteit te ontwikkelen, waardoor het uiteindelijke meer dan alleen een kopie is geworden.
